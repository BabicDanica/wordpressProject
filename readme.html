# 📘 WordPress ReadMe Ovaj fajl predstavlja statičku HTML stranicu (`readme.html`) koja dolazi kao deo WordPress instalacije. Služi kao vodič za korisnike koji žele da instaliraju, ažuriraju ili migriraju WordPress platformu. --- ## ✨ Šta je ovo? Fajl sadrži zvanična uputstva za: - Brzu instalaciju WordPress-a ("Famous 5-minute install") - Ažuriranje sistema (automatski ili ručno) - Migraciju sadržaja sa drugih CMS sistema - Sistemske zahteve (PHP, MySQL) - Linkove ka dokumentaciji, forumima i zajednici --- ## 📂 Struktura i tehnologije - `HTML5` – Osnovna struktura dokumenta - `CSS` – Stilovi se učitavaju iz WordPress admin direktorijuma - `wp-admin/install.php` – Link ka početnoj instalaciji - `wp-config.php` – Konfiguracija baze podataka --- ## 🛠 Sistemski zahtevi **Minimalno:** - PHP 5.6.20+ - MySQL 5.0+ **Preporučeno:** - PHP 7.4+ - MySQL 5.6+ - Apache mod_rewrite modul - HTTPS podrška --- ## 🔗 Korisni linkovi - 📚 [WordPress Codex](https://codex.wordpress.org/) - 📰 [WordPress Blog](https://wordpress.org/news/) - 🌐 [WordPress Planet](https://planet.wordpress.org/) - 💬 [Support Forums](https://wordpress.org/support/forums/) - 💻 [IRC kanal](https://make.wordpress.org/support/handbook/appendix/other-support-locations/introduction-to-irc/) --- ## 📜 Licenca WordPress je open-source softver pod [GNU GPL](https://www.gnu.org/licenses/gpl-2.0.html) licencom (verzija 2 ili novija).
